# Node REST API

> 
API CRUD REST completa usando Node.js sem estrutura

Isso é para fins de aprendizagem / experimentais. Na maioria dos casos, você usaria algo como Express em um projeto de produção

```
# Routes
GET      /products
POST     /products
GET      /products/:id
PUT      /products/:id
DELETE   /products/:id

```

## Uso

```
# Install dependencies
npm install
yarn install

# Run in develpment
npm run dev
yarn run dev

# Run in production
npm start
yarn start
```
